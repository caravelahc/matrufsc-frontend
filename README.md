# matrufsc-frontend

Caravela's MatrUFSC built using Elm.

# Setup
Install elm by using the [official guide](https://guide.elm-lang.org/install.html).

Compile Elm using

`elm-make Main.elm --output index.html`

or use `elm-reactor` and visit `http://localhost:8000`
